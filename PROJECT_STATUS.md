# Project Status - Trade Deals UK

## Overview
A UK Trade Deals & Price Comparison Engine focusing on promotions/discounts for building materials, tools, and power tools from retailers like Wickes, Toolstation, B&Q, etc.

## Tech Stack
- **Database**: Neon (Serverless PostgreSQL)
- **Backend/Sync**: TypeScript/Node scripts with GitHub Actions
- **Frontend**: Astro + TypeScript + Tailwind CSS
- **Hosting**: Vercel (free tier)
